# CSS Advanced Project - Smileschool

## Description
Ce projet est la suite du projet **HTML Advanced**. Ici, l'objectif est de styliser la page web créée dans le projet précédent, en suivant **exactement** le design fourni dans Figma.  
Le projet se concentre uniquement sur le CSS et la mise en forme visuelle de la page.

---

## Objectifs
- Importer et connecter un fichier CSS externe à votre page HTML.  
- Styliser la page section par section : Header, Hero, Quotes, Videos, Membership, FAQ et Footer.  
- Respecter **exactement** les dimensions, couleurs, typographies et positionnements du Figma.  
- Utiliser des **sélecteurs CSS simples et propres**.  
- Centraliser et réutiliser les styles communs lorsque c’est possible.

---

## Structure du projet
```
css_advanced/
│
├── index.html       # Page HTML copiée du projet 1
├── styles.css       # Fichier CSS contenant tous les styles
└── images/          # Dossier contenant toutes les images nécessaires
```

---

## Installation et utilisation
1. Cloner le dépôt :
```bash
git clone <URL_DU_DEPOT>
```
2. Ouvrir `index.html` dans un navigateur pour visualiser la page.  
3. Modifier `styles.css` pour ajuster le style si nécessaire.  

---

## Figma
Le design est fourni dans le fichier Figma.  
- Assurez-vous de dupliquer le fichier dans vos drafts pour accéder aux mesures exactes, aux couleurs et aux images.  
- Polices utilisées : `Source Sans Pro` et `Spin Cycle OT`.  
- Certaines valeurs sont en float, vous pouvez les arrondir pour simplifier le CSS.

---

## Notes importantes
- Ce projet **ne doit pas modifier le HTML** existant (il est copié depuis le projet HTML Advanced).  
- Le CSS doit être propre et clair, avec des commentaires si nécessaire.  
- Les images et icônes doivent être placées dans un dossier `images/` et reliées correctement dans le CSS et le HTML.
